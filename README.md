# Kamegg Homepage

Eine statische React-Website, gebaut mit Vite und Node.js-Tooling. Die Seite stellt Kamegg vor und enthält eine kompakte Veranstaltungsübersicht. Das Projekt ist für GitHub Pages oder anderes statisches Hosting geeignet.

## Entwicklung

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Das gebaute Ergebnis liegt in `dist/`.

## Vorschau

```bash
npm run preview
```

## Hinweis zu GitHub Hosting

Das Projekt wird automatisch über GitHub Actions auf GitHub Pages veröffentlicht.

Live unter:

https://stefankaltschik89.github.io/Kamegg_homepage/

Wenn du später eine echte Node.js-API willst, sollte sie separat gehostet werden, weil GitHub Pages nur statische Dateien ausliefert.
